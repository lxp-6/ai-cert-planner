# 艾宾浩斯遗忘曲线参考

本 Skill 默认使用艾宾浩斯遗忘曲线的关键复习间隔：

```
第 1 天 → 第 2 天 → 第 4 天 → 第 7 天 → 第 15 天
```

即一个新知识点在学完的当天算作第 0 天，之后在第 1、2、4、7、15 天进行复习。每次复习时长建议 5-15 分钟，以快速回忆、做题巩固为主。

## 为什么用这些间隔

艾宾浩斯实验指出，遗忘在学习后 20 分钟开始，1 小时后保留约 44%，1 天后约 34%，6 天后约 25%，31 天后约 21%。在遗忘尚未大幅下滑的时间点复习，能以最小时间投入换取最高记忆保持率。

## 在 Skill 中的实现

- `scripts/ebbinghaus_review.py` 为每个科目首次学习日期后推 1、2、4、7、15 天生成复习任务。
- 复习任务默认时长 10 分钟，优先级为 `medium`。
- 若复习日期已过今天或已过考试日期，则跳过该复习点。

## 自定义复习间隔

如需修改复习间隔，编辑 `scripts/ebbinghaus_review.py` 中的 `EBBINGHAUS_INTERVALS` 列表即可。
